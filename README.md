# RSG-Demos

Test-bed for games developed by **Radley Sustaire** together with **Claude Code (Anthropic)**. Every milestone of every game in active development is published here so the current state can be playtested from any browser, on any device, with no install.

**Live site:** https://radgh.github.io/RSG-Demos/

The root `index.html` is an auto-generated hub that links to each game's latest milestone build. This repo is a pure GitHub Pages publishing target — source code for each game lives in a separate workspace and is copied in via `deploy_pages.sh`.

---

## Games

### Roguelike (game2)
A browser-based 2.5D isometric roguelike with local co-op (1–4 players). Fast 3–10 minute runs, persistent progression, 9 classes with 5 abilities each, 200+ milestones of content including mythic items, endless dungeons, arena mode, crafting, and mercenaries.

- **Play:** https://radgh.github.io/RSG-Demos/game2/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game2/assets/

### Roguelike III (game3)
A third-person top-down twin-stick action RPG. Full conversion from the isometric Roguelike engine to fast-paced twin-stick combat with 7 classes, 360° aiming, dash/roll, and mobile virtual joystick support.

- **Play:** https://radgh.github.io/RSG-Demos/game3/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game3/assets/

### Spellroads (game4)
An open-world turn-based RPG set in a procedurally generated Norse world. Assemble a party of adventurers, travel the Spellroads — ancient magical highways — and restore them from growing darkness. 2–4 player local co-op, formation-based combat, deep progression.

- **Play:** https://radgh.github.io/RSG-Demos/game4/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game4/assets/

### Astral Crypts (game6)
A top-down action roguelite with procedural dungeon generation and 4 classes (Ironclad, Riftmage, Phantom, Voidpriest). Meta-progression, melee and ranged combat, boss encounters, equipment. Built with PixiJS v8 at 480×270 internal resolution scaled to 3×.

- **Play:** https://radgh.github.io/RSG-Demos/game6/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game6/assets/

### Wanderlight (game7)
A turn-based caravan RPG with branching dialogs. Lead a party of adventurers across a medieval fantasy world threatened by demonic corruption. Side-scrolling caravan travel, tactical grid combat, 12 skill trees, and context-sensitive dialog.

- **Play:** https://radgh.github.io/RSG-Demos/game7/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game7/assets/

### Vagrant & Ash (game8)
A mobile-first dark fantasy survival RPG. Lead a party of 1–4 across a dying continent. Point-buy character creation, permadeath, emergent storytelling. Every choice costs something.

- **Play:** https://radgh.github.io/RSG-Demos/game8/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game8/assets/

### Aethermoor Chronicles (game10)
A vibrant action RPG with loot, levelling, skill trees, mining and crafting. Fight enemies in real-time combat across a large open world.

- **Play:** https://radgh.github.io/RSG-Demos/game10/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game10/assets/

### Stormtown (game11)
Survive massive tornadoes and zombie hordes in a procedural city. Rescue Sam, collect power-ups, drive cars, dodge weather events, and escape by helicopter. 6 zombie variants, 6 map zones, night mode, ghost replay, achievements, XP progression, daily challenges.

- **Play:** https://radgh.github.io/RSG-Demos/game11/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game11/assets/

### Penny Panic (game12)
Two players, one phone. Split the screen in half — you own your side. Ten pennies fall into the void. Race your opponent to tap them first. Shoot bugs for bonus points. Miss one and lose a point. Highest score when the last penny is gone wins the round; first to two rounds takes the match.

- **Play:** https://radgh.github.io/RSG-Demos/game12/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game12/assets/

### Emberveil (game13)
A high fantasy party RPG. Build a party of up to 8 heroes and companions and journey across four acts — goblin frontier, volcanic wastelands, the Shattered Hell, and the Cosmic Void — in cinematic auto-play combat. 14 playable classes, procedurally generated equipment, deep progression.

- **Play:** https://radgh.github.io/RSG-Demos/game13/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game13/game-info/

> **Note:** The "Play" link for Emberveil goes to the **marketing front page**, not directly to the game. Click **"Play Now"** on that page to launch the actual game at `/game13/play.html`. This split is intentional — the front page is used for screenshots, feature overviews, and external sharing, while `play.html` is the runtime game entry.

### HOOPSHOT (game14)
Slingshot basketball vs. CPU. Drag to aim, release to launch, and out-score the computer across 9 progressively wilder rounds with shrinking, moving hoops and wind. Touch and mouse supported. PixelLab art, OpenAI backgrounds, ElevenLabs sound.

- **Play:** https://radgh.github.io/RSG-Demos/game14/
- **Game info / assets:** https://radgh.github.io/RSG-Demos/game14/assets/
