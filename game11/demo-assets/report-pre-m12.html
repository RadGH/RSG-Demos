Milestone Plan
M12 Ghost Replay System
⚡⚡⚡⚡⚡
High
  • Record player path every 0.5s during each run: array of {x, y, t} position snapshots
  • Persist last-run path in localStorage under stormtown_ghost (compressed, max 500 points)
  • On next run, spawn ghost sprite: semi-transparent white silhouette, depth 8
  • Ghost follows recorded path with linear interpolation between points, time-synced to run clock
  • Ghost death moment: when run clock hits the time the player died last run, ghost flashes and dissolves with particle burst
M13 Survivor Chain
⚡⚡⚡⚡⚡
High
  • Rescued bonus survivors physically follow player in a trailing chain after rescue
  • Follow AI: lerp each survivor toward the entity ahead in chain at 80% player speed
  • Chain penalty: each follower slows player by 5px/s (max ~25px/s with 5 survivors)
  • Sacrifice mechanic: player can walk chain into a tornado — survivors count as 0 HP deaths, tornado briefly steers toward chain
  • Survivor sprites render behind player (depth below player), show distress animation when tornado is near
M14 Achievement System
⚡⚡⚡⚡⚡
Must-Ship
  • 12 achievements stored in stormtown_achievements localStorage key
  • Examples: "First Escape", "Speed Runner" (<60s), "Pacifist" (0 kills), "Full House" (all 3 survivors), "Untouchable" (no damage taken), "Tornado Dodger" (survived 4 tornadoes), "Mechanic" (drove a car), "Last Breath" (escaped at 1HP)
  • Achievement pop-up: slides in from top-right corner, gold badge style, 2.5s auto-dismiss
  • Achievement gallery tab on title screen — shows locked/unlocked state with unlock date
  • First-run achievements unlock without localStorage (session-only fallback)
M15 Night Mode / Dynamic Lighting
⚡⚡⚡⚡⚡
Stretch
  • After 90s game time, day gradually transitions to night (30s blend)
  • Night: dark overlay (depth 35) fades from alpha 0 → 0.7 over transition window
  • Player light radius: 4 screen-edge fog-of-war panels leaving ~200px clear zone around player
  • Torch items on map: pick up to extend light radius to ~320px for 30s
  • Buildings gain warm lit-window highlights at night (small amber rectangles on facades)
  • Tornadoes emit ghostly pale blue light ring at night (additive blend circle)
M16 Score Meta-Progression
⚡⚡⚡⚡⚡
Must-Ship
  • Persistent XP: escape=50, each survivor rescued=15, each zombie kill=1, survival time=1 per 10s
  • XP accumulates across runs in stormtown_xp localStorage key
  • Levels 1–20 with thresholds; each level unlocks a cosmetic or badge shown on title screen
  • Unlocks include: player sprite tints (blue at Lv5, red at Lv10, gold at Lv15), "veteran" title badge
  • Title screen shows: "Lv.7 VETERAN — 2340 XP" with progress bar to next level
  • End-run screen shows XP earned this run + total + level-up animation if applicable

Milestone Risk Level Mitigation
M15 Fog-of-war via 4 screen panels leaves diagonal corners partially lit — will look wrong High Use Phaser RenderTexture mask with radial gradient cutout instead of edge panels
M12 500-point ghost path at ~150 bytes/point = 75 KB per ghost — may exceed localStorage quota on older Safari Med Delta-compress path (store only dx/dy changes), implement graceful fallback if quota exceeded
