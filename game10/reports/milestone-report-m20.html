Spotlight
Standout Features
🌀
ZoneEvents — Dynamic World Events
Four rotating world events shake up each play session: Double Loot (all drops ×2), Elite Surge (50% of enemies become elites), Mining Rush (ore respawn instantly) and Monster Horde (rapid enemy spawns). Events announce via a HUD banner and last 3 minutes each. Cycle repeats so every hour of play feels different.
🎵
AudioManager — Zero-File Procedural Audio
The entire audio system uses Web Audio API oscillators, filters, and envelope shaping — no audio files, no CDN, no loading cost. Lazy-initialises on first user gesture to satisfy browser autoplay policies. Sounds are tuned per action: sword swing uses a detuned sawtooth with a short decay; boss roar layers a low square wave with a noise burst; level-up plays a triumphant major chord cascade.
🐉
The Ancient One — 4-Phase Final Boss
The Void's final guardian scales through four distinct phases at 75%, 50%, and 25% HP thresholds. Phase 1: standard melee + projectile spiral. Phase 2: clones appear. Phase 3: shadow dashes. Phase 4: enrage — after 4 minutes real-time the boss enters permanent rage mode, dealing 3× damage. Epic and Legendary loot is guaranteed on kill.
🏆
Achievement System
15 achievements track long-term milestones: First Blood, Level 10, Zone Explorer, Combo Master (x10), Boss Slayer, Legendary Find, Master Crafter, and more. Each unlock triggers a slide-down toast with animated icon and gold border. Progress persists in localStorage and awards player titles shown in HUD.

Quality
Bugs Fixed & Known Issues
🖱
Mouse clicks not registering on menu buttons Fixed M20
Root cause: #ui-root had pointer-events:none for game-world passthrough. MainMenuUI's container div had no override, so all mouse events fell through to the canvas. Fixed by adding pointer-events:auto to the menu container and focus() on the default button for keyboard accessibility.
🗂
Asset 404s on GitHub Pages Fixed M15
Absolute /sprites/ paths worked on localhost but resolved to the root of radgh.github.io rather than the /game10/ subdirectory. Migrated all paths to ./sprites/ relative form.
📱
Touch joystick can overlap skill hotbar on small phones Known
On screens narrower than 360px the virtual joystick and the skill buttons share the same vertical band. A future layout pass should move hotbar to the top-right in that case.
🔊
Audio silent until first interaction on iOS Safari By Design
AudioManager lazy-initialises the AudioContext on first pointerdown event as required by browser autoplay policy. No workaround needed — this is correct behaviour.
Looking Ahead
Suggestions & Ideas
Strength
The procedural audio system is a standout — no audio files, zero loading overhead, and surprisingly expressive. Worth keeping this approach for any future audio additions.
Idea — Seasonal Events
The ZoneEvents foundation could support "seasonal" events (one per real-world day) that persist across sessions. A daily "Ancient Rift" event that spawns unique enemies for 24 hours would drive return engagement.
Idea — Leaderboard
A simple localStorage-backed highscore board (most enemies killed, deepest zone reached, fastest boss kill) shown on the main menu would give long-term players a goal to chase.
Performance Watch
At 185 KB the bundle is comfortable, but the particle system and Y-sort pass both run every frame. If enemy count scales beyond 30 on-screen, profile and consider spatial bucketing for the sort.
Idea — NG+ Mode
After defeating The Ancient One, a New Game+ mode with enemies at 2× stats and a chance at exclusive Legendary variants would meaningfully extend the loot-chase loop.
