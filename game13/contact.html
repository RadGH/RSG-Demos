About the Creator

Radley Sustaire is a developer based in Veneta, OR who has been building web applications for over a decade. Emberveil began as an experiment in AI-assisted game development — using Claude Code (Anthropic) to design, implement, and iterate on a full-featured party RPG across dozens of milestones. The project spans custom combat systems, procedural loot, an original dark-fantasy world, and 20 playable hero classes, all created collaboratively with AI tooling.
