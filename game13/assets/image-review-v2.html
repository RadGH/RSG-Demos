Image Review V2

Live image catalog. Select images you want to act on, then choose how to send them to Claude.

Approving images is a two-step flow. Selecting cards on this page does not persist anything — the page is read-only and your selections live only in this tab. To actually approve:
  1. Select the cards you want to approve.
  2. Click Copy approve command (or Copy as list + tell Claude "approve these").
  3. Paste into Claude. Claude runs node scripts/approve-image-review-v2.cjs <ids…>, which copies the candidate file into the live path and marks the entry approved. Pending status disappears on the next page reload.
Tip: Copy approve command produces a ready-to-run shell line with the script and IDs already filled in. Approve all covers the entire open batch in one command.
