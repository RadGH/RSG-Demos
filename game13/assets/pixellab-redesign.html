Tools · PixelLab Redesign

Reference sheets, candidate frames, and approval status for the BitForge-only character redesign pipeline.

Canonical spec lives in /docs/image-policy.md — 7 poses, 256×256, no animation files. The /assets/ gallery is stale and may show incorrect frames; treat the docs as source of truth.
Generator entry points: node scripts/pixellab/gen-reference-sheet.cjs <id>, node scripts/pixellab/gen-frame.cjs <id> <pose>, node scripts/pixellab/approve.cjs <id>. State file: public/data/pixellab_redesign_state.json. Banned-substring guardrail lives in each character's bannedSubstrings field in public/data/art_direction/.
