Custom Content Loader

Upload a JSON pack to add skills, classes, items, events, appearances, characters, or loot tables. Packs are merged at runtime; the game never hardcodes around them.

Paste a pack, single entity, or bare array. Kind is inferred from shape when no id/version wrapper is present.

Download the JSON Schema v1 files. Point your editor (VS Code, JetBrains) at these via $schema for completion and validation.

Reference packs that exercise every DSL op and pack kind.
