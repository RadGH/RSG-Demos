How to use this page

Tile actions on every image card:

  • Arc (Archive) — hides the image from active sections; reversible via the Archived / Unattached section. Local UI flag only — no file is moved.
  • Lib (Library) — flags the image as moved to the unassigned-appearance library. Reversible by un-toggling.
  • Del (Delete) — flags the image for deletion. Reversible until you actually run the generated prompt against the project; nothing is deleted from disk by clicking the button.

Selecting: click a tile (or the checkbox left of a sprite name) to add it to the working selection. The selection editor opens at the bottom-right; assign an action there to apply it to every selected image at once.

Quick Select (footer): just gives you a paste-ready list of the selected file paths — for handing off images to Claude with no action attached.

Generate Prompt (footer): produces the full action-batch prompt for every flagged image.
