Act I — What the Engine Does Today

The combat round, in one picture

build turn order → tick statuses → regen MP → for each unit { skip-if-dead → tick CD → stun? → AI → extraAction? } → check victory/defeat

Main loop: CombatScreen.js:990-1052. Headless twin: simulator.js:391-525. Any change to one must port to the other or the simulator stops predicting the UI.

Damage pipeline

raw → hit → block → mitigation → AoE falloff → crit → dmgReduct → dmgBuff → steal → final

Hit: clamp(actor.hit − target.dodge, 5, 95)   Crit: min(75, 5 + critPct + critChance×100) at 1.5 + critDamage×

Mitigation (M84): dmg = max(0, raw − armor)   — flat, not percentile. True damage skips entirely.

AoE falloff: 1→100%, 2→80%, 3→60%, 4+→50%.

The bug class we want to kill

Strength is spelled four different ways depending on where you're reading from. If a mod author ships { stat: "strength", value: 2 } it silently deals 0 damage bonus because the affix loader expects "str". We can fix this once with a normalizing loader and schema validation.

Status effects: the dual model

DoTs/CC live in unit.statuses[] as { type, duration, power }. Buffs like dmgBuff, critBonus, armor, magicResist live as top-level fields with paired *Rounds counters. Two code paths, two tick sites, one source of confusion.

    Other suggestions

    • Unify the status model. Move dmgBuff, critBonus, etc. into statuses[] so there's one tick site, one serialization path.
    • Per-skill cooldown map. actor.cooldowns[skillId] replaces the single skillCooldown slot — unlocks mod-authored characters with 4+ active skills.
    • Schema files live under /schemas/v1/ and are loaded by the Custom Content page via $schema URLs — mods become portable across versions.
    • Seeded RNG for combat — the simulator already supports it; expose the seed so mod authors can share reproducible "tricky encounter" bug reports.
    • Telemetry of shim hits — every time the legacy-name loader fires, count it; dashboard shows which mods need cleanup.
    • Diff-view for balance — when a mod character/spell is loaded, Combat Simulator shows side-by-side DPS vs. nearest vanilla equivalent so you can flag outliers.
