AI Content Generator

Describe a skill, class, character, item, loot table, event chain, or companion in plain English. Claude writes valid JSON against /schemas/v1/. Paste the result into Custom Content Loader.

How to use
  1. Paste your Anthropic API key below and click Save — it's only stored in your browser's localStorage.
  2. Pick what to generate (skill, class, item, etc.) and a model.
  3. Describe what you want in plain English and click Generate JSON.
  4. Lint the result, then click Open Loader → to load it into the running game.

Need a key? Get one at console.anthropic.com. Uses claude-haiku-4-5 by default for speed.

How it works

On Generate, the tool fetches the relevant schema(s) from /schemas/v1/, builds a system prompt that includes the schema text plus the DSL op list, and calls the Anthropic Messages API directly from your browser (using the anthropic-dangerous-direct-browser-access header). The model returns JSON only. We strip any markdown fences, JSON-parse, and show the result. Your key never leaves your browser except when it's sent to api.anthropic.com over HTTPS.
