Tools · Character Redesign

Rebuilt reference sheets per class using tighter art-direction prompts, grouped by gender. Each row shows the old OpenAI portrait (the original identity anchor) alongside the new male and female SpriteCook references so you can flag miscategorized genders or regressions at a glance.

Pipeline: prompts live in public/data/art_direction/<id>.json; references are generated via the SpriteCook MCP (generate_game_art, pixel mode, 256×256) and saved to public/images/pixellab/<id>/reference.png. The canonical pose spec is in /docs/image-policy.md (7 poses per character). Sub-sprite frames below each card are optional; use the floating toggle to hide them while auditing the main references.
